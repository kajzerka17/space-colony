The app is mainly optimized for the 'Medium Phone' in Android Studio in terms of button placements, however everything works and was visible in the actual phones we tested (some xiaomi and samsung phones).

We used Android Studio for programming, Pixelmator Pro for creating crew icons.

Instalation: You can download the repository and compile the app on your own in Android Studio or download the ready apk. file.

About the game: This game is about managing crew and power. The game is split into days and the player decides what to do each day. The player have to complete one random mission each day to progress. Difficulty of the game increases with each day, so good managment is necessary to progress with the game. The player can choose from a variety of different crew types and upgrades. Game is virtually infinite.

Bonuses: We have all of them
